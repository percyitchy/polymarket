Проблема: CLOB API Polymarket (https://clob.polymarket.com/markets/{condition_id}) возвращает HTTP 404 для закрытых/удаленных рынков, поэтому мы не можем получить текущую цену.

Текущие fallback методы:
1. CLOB API (основной) - возвращает 404 для закрытых рынков
2. HashiDive API - временно недоступен
3. Средняя цена из wallet_prices - работает только если есть данные о сделках
4. Polymarket data-api - не всегда доступен

Вопрос: Где еще можно получать данные о ценах рынков Polymarket, особенно для закрытых рынков?

Нужно:
- condition_id (hex string, 66 символов)
- outcome_index (0, 1, 2...)
- price (float, 0.0-1.0)
- timestamp (опционально)

Требования:
- Публичный/бесплатный API
- Поддержка запросов по condition_id
- Низкая задержка (<5 сек)
- Желательно исторические данные

Интересуют:
1. Альтернативные API endpoints Polymarket (GraphQL, WebSocket?)
2. Сторонние сервисы, агрегирующие данные Polymarket
3. Публичные индексаторы блокчейна (The Graph?)
4. Источники исторических данных о закрытых рынках

Пожалуйста, предложите конкретные решения с примерами URL и форматов запросов.
